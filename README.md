🌤️ SkyCast

SkyCast is a simple React weather app that shows real-time weather information for any city, including temperature, humidity, and conditions.

🚀 Features

🌍 Search weather by city
🌡️ Temperature display
💧 Humidity info
🌥️ Weather condition icons
📱 Responsive design
🛠️ Tech Stack

React.js
JavaScript
CSS
Weather API (OpenWeather)

⚙️ Run Locally
git clone https://github.com/SumitSabde/skycast-wheather-prediction-app.git
2.cd skycast-wheather-prediction-app
npm install
npm start

App runs on http://localhost:3000
